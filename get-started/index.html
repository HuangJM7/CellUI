# 快速开始

# 介绍

这是我在学习 Vue 过程中做的一个 UI 框架，希望对你有用。

# 开始使用

# 1. 添加 CSS 样式

使用本框架前，请在 CSS 中开启 border-box

*,*::before,*::after{box-sizing: border-box;}

IE 8 及以上浏览器都支持此样式。

# 2. 安装 unit-ui

npm install unit-ui

或

yarn add unit-ui

# 3. 引入 unit-ui

# 3.1完整引入

在 main.js 中写入以下内容：

import UnitUI from 'unit-ui'
import 'unit-ui/dist/index.css'
Vue.use(UnitUI);

new Vue({
 el: '#app',
 render: h => h(App)
});

# 3.2局部引入

在 main.js 中写入以下内容：

import {Button, Icon} from 'unit-ui'
import 'unit-ui/dist/index.css'

export default {
  name: 'app',
  components: {
    'u-button': Button,
    'u-icon': Icon
  }
}
